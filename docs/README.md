- 게임시작
    -[x] 게임 시작 문구 출력
    -[x] 다리 길이를 입력 받는다
        - 3~20이 아닐 경우 예외 처리

- 다리 생성
    -[x] 다리를 생성한다

- 이동 방향
    -[]U or D로 이동방향을 입력받는다
        - U or D 입력이 아닐 경우 예외처리
    -[] 입력한 값이 U라면 위로 이동
    -[] 입력한 값이 D라면 아래로 이동

- [] 이동에 성공하면 O 표시
    - 이동 반복

- [] 이동에 실패하면 방향에 X 표시
    - 재시작 여부 조사
        -[] 입력한 값이 R이라면 게임 재시작
        -[] 입력한 값이 Q라면 게임 종료
            - R or Q가 아니라면 예외처리

-[] 게임 성공 여부 출력
-[] 총 시도한 횟수 출력
